# Period Comparison Template

## Structure

### Comparison Parameters
- Current period: [dates]
- Prior period: [dates]
- Comparison type: WoW / MoM / YoY
- Day-of-week aligned: Yes/No

### Summary Table
| Metric | Current | Prior | Delta | Delta % | Status |

### By Channel (if multi-channel)
Separate table per channel, same format.

### By Geo
| Metric | NA Current | NA Prior | NA Delta % | INTL Current | INTL Prior | INTL Delta % |

### Key Takeaways
- 3-5 bullets: biggest changes, likely causes (from hypothesis agent), actions needed
