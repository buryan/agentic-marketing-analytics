# Weekly Report Template

## Structure

### Executive Summary
- 3-4 bullet TL;DR covering: overall direction, biggest win, biggest concern, one action item
- Traffic light status for each channel analyzed

### Channel Performance
One table per channel with: Metric | Current Week | Prior Week | WoW Delta % | YoY Delta % | vs Plan | Status

### Anomalies Detected
Table of any metrics exceeding z-score threshold. If none: "No anomalies detected this week."

### Hypotheses
Why did significant moves happen? Table from hypothesis agent output.

### Recommended Actions
ICE-scored table from synthesis agent (if multi-channel) or channel agent.
Maximum 5 actions. Each with: Action | ICE Score | Owner | Expected Impact.

### Data Quality Notes
Any WARN items from data quality validation. Any caveats on the analysis.

### Memory Updates
What was updated in /memory/ after this analysis run.
